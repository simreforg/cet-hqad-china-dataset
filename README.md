# cet-hqad-china-dataset
This repository contains the processed datasets supporting the study:

**"The Impact of Carbon Emissions Trading Pilot Policies on High-Quality Agricultural Development in China."**

The data were used to evaluate the effects of China's Carbon Emissions Trading (CET) pilot policies using causal inference and machine-learning methods.

---

## Repository Contents

panel_data_processed.csv  
Processed province–year panel dataset used in the empirical estimation.

---

## Data Sources

The dataset was constructed from publicly available statistical sources:

- China Statistical Yearbook  
- China Rural Statistical Yearbook  
- China Agricultural Yearbook  
- Provincial statistical yearbooks  

These files represent **processed research data used to generate the empirical results reported in the paper.**
